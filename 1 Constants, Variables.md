# Constant/Variables

Constants are *immutable*. They are fixed.

```swift
let myName = "Grant"
```

```swift 
let age = 25
age = 30 // ❌ Error: Cannot change a constant
```

Variables are *mutable*, or able to be changed.

```swift
var myJob = "Pilot"
```
